Here are the steps admin needs to follow to integrate to any WordPress site.

1) Make Sure the WordPress website has the ACF pro plugin and if it's not available then you install it.
2) Import the ACF fields by importing the JSON file in ACF >> Tools >> Import fields.
3) Install plugin by uploading the plugin zip file in WordPress plugin admin settings.
4) Once you activate the plugin there will post type as “Scooters”.
5) In Scooter admin needs to add the respective scooters.Admin needs to enter following fields:
=> Title
=> Featured Image
=> ACF field groups.
6) Range Calculator Settings page
